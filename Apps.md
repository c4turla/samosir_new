SAMOSIR - Backend & Web Frontend Documentation
📋 Deskripsi Singkat

SAMOSIR adalah sistem informasi untuk Pelabuhan Perikanan Nusantara Sibolga yang menangani manajemen kapal, keberangkatan, kedatangan, dan layanan jasa pelabuhan.
🏗️ Tech Stack

    Backend: Laravel 12 + MySQL

    Frontend Web: Inertia.js + Vue 3 + Tailwind CSS

    Authentication: Laravel Sanctum (API) + Session (Web)
